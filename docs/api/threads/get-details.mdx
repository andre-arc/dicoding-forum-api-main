# View Thread Details

`GET /threads/{threadId}`

Fetch thread details from the forum.

```yaml title="Path Params"
threadId: thread-unique_id
```

### Response

<Success>200 OK</Success>

```json title="Simplified Response Body"
{
  "status": "string",
  "data": {
    "thread": {
      ...Thread,
      "comments": [
        {
          ...CommentOrReply,
          "likeCount": "int",
          "replies": [
            CommentOrReply,
            CommentOrReply
          ]
        }
      ]
    }
  }
}
```

```json title="Thread"
{
  "id": "string",
  "title": "string",
  "body": "string",
  "date": "date",
  "username": "string"
}
```

```json title="CommentOrReply"
{
  "id": "string",
  "username": "string",
  "date": "date",
  "content": "string"
}
```
